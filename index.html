FAU Capstone Project — NYPD Arrest Data

This site hosts all deliverables for the Florida Atlantic University Capstone project that focuses on mapping and predicting offenses using NYPD arrest data.

This project identifies and predicts the most prominent offense types for each precinct for each day of the week to support public-safety planning.
